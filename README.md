# gym-socioTEQ
OpenAI Gym Style SocioTEQ Environment.

```
observation [7, 15, 86, 28]
action 3
new observation [7.7, 16.5, 94.6, 30.8]
reward 37.4
done 0

```

# Requirement

Python >= 3.5

# Install
```
git clone https://github.com/efotopoulou/gym-socioTEQ.git
cd gym-socioTEQ/
pip install -e .
```
# Try example agents
```
cd examples/
python random_agent_SocioTEQEnv.py
```
